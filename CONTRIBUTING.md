# Contributing to Chainproof

Thanks for helping make software supply chains safer!

## What fits

- New security checks or verification patterns
- Better scripts for dependency auditing
- Configurations for additional tools
- Corrections or clarifications in the handbook
- Translations

## How to contribute

1. Fork the repo
2. Create a feature branch: `git checkout -b feat/your-idea`
3. Commit changes with a signed commit
4. Push and open a PR

## Guidelines

- Scripts must work on Linux and macOS; Windows via Git Bash
- Test with `shellcheck` for shell scripts
- Keep README copy-pasteable — commands should work as-is
- Reference security tools by their official sources
